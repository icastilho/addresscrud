Hello!!! This is a Address CRUD Example


You need a Restful client to access the examples

try this one for Fire Fox

You may also have to set the Content-Type header of the request to application/json.

Urls

  • Find All - GET - https://addresscrud.herokuapp.com/address
  • CREATE - POST - https://addresscrud.herokuapp.com/address
  • Get Address - GET - https://addresscrud.herokuapp.com/address/{id}
  • UPDATE - PUT - https://addresscrud.herokuapp.com/address/{id}
  • DELETE - DELETE - https://addresscrud.herokuapp.com/address/{id}
